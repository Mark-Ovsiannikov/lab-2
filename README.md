# Flappy Bird Unity

**Flappy Bird Unity** — це проста 2D-гра на Unity, у якій гравець керує пташкою, що летить між перешкодами. Основна мета гри — пролетіти якомога далі, не врізатися в труби та набрати максимальну кількість очок.

## Опис гри

У грі пташка постійно падає вниз під дією гравітації. Гравець повинен натискати ліву кнопку миші або клавішу `Space`, щоб пташка підстрибувала вгору.

На шляху пташки з’являються труби. Вони рухаються з правої частини екрана вліво. Гравець повинен проводити пташку між верхньою та нижньою трубою. Якщо пташка торкається труби або іншої небезпечної перешкоди, гра завершується.

## Як грати

1. Запустити сцену гри.
2. Натискати ліву кнопку миші або `Space`, щоб пташка підстрибувала.
3. Пролітати між трубами.
4. За кожний успішний проліт між трубами гравець отримує очко.
5. Якщо пташка врізається в трубу, землю або іншу перешкоду, гра завершується.
6. Після програшу рівень перезапускається або відкривається екран меню.

## Керування

| Дія | Кнопка |
|---|---|
| Стрибок / підліт пташки | Ліва кнопка миші |
| Стрибок / підліт пташки | Space |

## Основні об’єкти гри

### Пташка

Пташка є головним персонажем гри. Вона має компонент `Rigidbody2D`, завдяки якому на неї діє гравітація. Коли гравець натискає кнопку, пташка отримує імпульс вгору.

Основні функції пташки:

- реагує на натискання миші або клавіші `Space`;
- підстрибує вгору;
- падає вниз під дією гравітації;
- взаємодіє з перешкодами;
- отримує очки при проходженні між трубами;
- завершує гру при зіткненні з небезпечною зоною.

### Труби

Труби є головними перешкодами у грі. Вони з’являються попарно: одна труба зверху, інша знизу. Між ними є проміжок, через який повинна пролетіти пташка.

Основні функції труб:

- рухаються з правого боку екрана вліво;
- створюють перешкоду для гравця;
- мають колайдери для перевірки зіткнення;
- між трубами розташована зона для нарахування очок.

### Зона очок

Між верхньою та нижньою трубою розміщується невидимий об’єкт із тегом `Score`. Коли пташка пролітає через нього, гравець отримує одне очко.

### Зона програшу

Труби, земля або інші небезпечні об’єкти мають тег `Finish`. Якщо пташка торкається такого об’єкта, гра завершується.

## Ігрова логіка

Під час запуску гри пташка знаходиться на стартовій позиції. Гравець натискає ліву кнопку миші або `Space`, і пташка отримує силу вгору. Після цього вона знову починає падати вниз через гравітацію.

Труби постійно рухаються вліво. Гравець повинен вчасно натискати кнопку, щоб пташка проходила через отвір між трубами. Якщо пташка успішно проходить між трубами, рахунок збільшується.

Якщо пташка торкається перешкоди, вона перестає реагувати на натискання, програється звук зіткнення, а через короткий час сцена перезапускається або відкривається меню.

## Основні скрипти

### PlayerScript.cs

Скрипт `PlayerScript.cs` відповідає за керування пташкою.

Основні функції:

- перевіряє натискання лівої кнопки миші або клавіші `Space`;
- викликає стрибок пташки;
- скидає поточну швидкість перед стрибком;
- додає силу вгору через `Rigidbody2D`;
- програє звук стрибка;
- перевіряє зіткнення із зонами `Score` та `Finish`;
- нараховує очки;
- завершує гру після зіткнення з перешкодою.

Основні параметри в Inspector:

| Параметр | Опис |
|---|---|
| Jump Force | Сила стрибка пташки |
| Au Clip | Масив звуків для стрибка та програшу |
| Fire | Додатковий візуальний ефект під час стрибка |

### GameManager.cs

Скрипт `GameManager.cs` відповідає за загальний стан гри.

Основні функції:

- зберігає кількість очок;
- оновлює UI рахунку;
- контролює стан гри;
- може відповідати за перезапуск сцени або перехід у меню.

### PipeSpawner.cs

Скрипт `PipeSpawner.cs` може використовуватися для автоматичного створення труб.

Основні функції:

- створює пари труб через певний проміжок часу;
- задає випадкову висоту появи труб;
- контролює складність гри через швидкість або частоту появи перешкод.

### PipeMove.cs

Скрипт `PipeMove.cs` може використовуватися для руху труб.

Основні функції:

- рухає труби вліво;
- видаляє труби, коли вони виходять за межі екрана;
- не дає сцені накопичувати зайві об’єкти.

## Налаштування пташки

На об’єкті пташки повинні бути такі компоненти:

```text
Rigidbody2D
Collider2D
AudioSource
PlayerScript
```

Рекомендовані налаштування `Rigidbody2D`:

```text
Body Type = Dynamic
Gravity Scale = 1 або 2
Mass = 1
Linear Damping = 0
Freeze Rotation Z = увімкнено
```

Рекомендовані налаштування `PlayerScript`:

```text
Jump Force = 7
Au Clip Element 0 = звук стрибка
Au Clip Element 1 = звук зіткнення / програшу
```

## Налаштування труб

Кожна труба повинна мати:

```text
Sprite Renderer
Box Collider 2D
Tag = Finish
```

Між трубами потрібно створити невидимий об’єкт для очок:

```text
Box Collider 2D
Is Trigger = увімкнено
Tag = Score
```

## Налаштування фону

Фон, гори, хмари та інші декоративні елементи повинні бути позаду основних об’єктів.

Рекомендовані `Order in Layer`:

```text
Фон = -10
Гори / хмари = -5
Земля = 0
Труби = 2
Пташка = 5
UI = 10
```

## Асети

Для гри використовуються прості 2D-асети:

- спрайт пташки;
- спрайти труб;
- фон неба;
- гори або хмари;
- земля;
- звуки стрибка;
- звук зіткнення;
- звук програшу.

## Умови отримання очок

Гравець отримує очко, коли пташка успішно пролітає між трубами та торкається невидимої зони з тегом `Score`.

## Умови програшу

Гра завершується, якщо пташка:

- торкається труби;
- падає на землю;
- торкається об’єкта з тегом `Finish`.

Після програшу пташка більше не може стрибати, програється звук зіткнення, а сцена перезапускається або відкривається меню.

## Підсумок

**Flappy Bird Unity** — це проста аркадна 2D-гра, побудована на механіці стрибка та уникнення перешкод. Гравець керує пташкою за допомогою лівої кнопки миші або клавіші `Space`, пролітає між трубами та отримує очки за кожний успішний проліт. Гра тренує реакцію, точність і вміння контролювати рух персонажа.
